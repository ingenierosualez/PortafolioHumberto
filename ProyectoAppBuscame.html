App Móvil Búscame

Propósito del Software

Buscame es un sitio web y una aplicación móvil para Android versión 4.1+, en la cual se encuentran los negocios (de ixtlán del río, Nayarit), información de contacto, mapa de localización, los productos que ofrece al público pudiendo ser comidas, bebidas o artículos. De la misma manera se encuentran los servicios y las promociones que ofrece cada uno de los negocios registrados en la plataforma. Están categorizados lo que hace encontrarlos de una manera rápida, añadiendo el menú que ofertan al público, así como existe la opción de buscar de manera general tanto a los negocios como a lo que ofrecen por su nombre.



Tecnologías Utilizadas

HTML CSS JS jQuery PHP Laravel 5.7 MySQL Dart Flutter Api Rest
